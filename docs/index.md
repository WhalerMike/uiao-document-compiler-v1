# UIAO Documentation Site

Welcome to the Unified Integration Architecture Operations (UIAO) documentation portal.

## Quick Start

| Resource | Description |
|----------|-------------|
| [Document Library](documents/index.md) | Browse all UIAO documents with status badges |
| [Contributor Guide](contributing.md) | How to add or update documentation |
| [Request Export](request-export.md) | Request PDF or Word exports of any document |

## Operating Environment

This site is built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) and deployed via GitHub Pages from the [uiao-docs](https://github.com/WhalerMike/uiao-docs) repository.

## Site Conventions

- **Document Identifiers** follow the `UIAO-xxx` pattern
- **Status Badges** indicate document maturity: Current, Needs Replacing, Needs Creating
- **Download Buttons** appear at the top of every content page
- **Dark Mode** is available via the toggle in the header
