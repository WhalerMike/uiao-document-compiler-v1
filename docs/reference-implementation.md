# UIAO-Core Reference Implementation  
## “ATO in a Box” Starter Kit Architecture

This document describes the reference implementation that demonstrates the full UIAO continuous evidence pipeline in a minimal, reproducible environment.

---

# 1. Purpose

The reference implementation (“ATO in a Box”) provides:

- A **deployable example environment**  
- A **working KSI validation pipeline**  
- A **Quarto ATO dashboard**  
- A **minimal workload** demonstrating UIAO principles  

It is designed for agencies, integrators, and auditors who need a **concrete, runnable** example of UIAO in practice.
