# UIAO Single Source of Truth (SSOT)

The UIAO SSOT is the authoritative origin for all identity, addressing,
enforcement, telemetry, and governance objects. All other representations
are pointers, not copies. SSOT is the foundation of the Governance OS,
the root of the Provenance Chain, and the baseline for Drift Detection.

This file is the canonical SSOT definition. All UIAO components must
reference this file as the authoritative source.
